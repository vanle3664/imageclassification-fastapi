# imageclassification-fastapi

Step to run:

1. Install necessary library

```bash
   pip install fastapi
   pip install "uvicorn[standard]"
   pip install retina-face
```

2. Run it:

```bash
   uvicorn server:app --reload
```

3. Go to fastAPI web UI:
   http://127.0.0.1:8000/docs#/default/

   Click "Try it out"

   Upload image file

   Execute
